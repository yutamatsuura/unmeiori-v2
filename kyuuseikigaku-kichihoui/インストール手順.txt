□必要ソフトウェア
https://nodejs.org/en/
=>Recommended For Most Users

https://git-scm.com/downloads

□インストール
powershell(管理者）
cd ソースディレクトリ
npm install -g --production windows-build-tools
npm install webpack webpack-cli -g
npm install

□起動
./debug.bat

http://localhost:3001/ban_top_full.html



□リリース方法
./release.bat
※jsファイルがreleaseフォルダ内へと生成される。
生成されたrelease/js/ban.jsをFFFTP等でサーバーへと移動させる。
